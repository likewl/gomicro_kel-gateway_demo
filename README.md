# gomicro_kel-gateway_demo
gomicro加简易gateway实现web和rpc访问

# 用到的工具
- gomicro

	- 实现web和rpc
	- web用到了gin框架
	- rpc基于grpc+protobuf

- 自写简易kel-gateway

	- 网关

- consul

	服务注册与发现
